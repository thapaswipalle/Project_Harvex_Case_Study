## Executive Summary
As Team Leader for Project Harvex, I managed the end-to-end data strategy and relational architecture for a patented platform optimizing agricultural supply chains. My role spanned database normalization, data pipeline optimization, and exploratory analysis to guide executive business decisions.

## Technical Framework
* **Database Management:** MySQL
* **Analytics & Engineering:** Python, Java
* **Interface & Web Support:** HTML, CSS
* **Version Control:** Git

## Key Contributions

### Relational Database Architecture
* Engineered a fully normalized MySQL database structure featuring 8+ core tables to track supplier, inventory, and consumer metrics.
* Conducted performance tuning and query optimization that reduced backend data latency by 40%.

### Exploratory Data Analysis & Reporting
* Maintained data pipelines to perform exploratory analysis on customer purchasing behavior, identifying seasonal demand shifts and top-performing product categories.
* Automated data reporting workflows to assist internal stakeholders with real-time supply chain adjustments and inventory control.
